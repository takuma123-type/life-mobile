# LIFE - 機能一覧

## 📱 基本機能

### 1. スプラッシュ画面
- アプリ起動時に2秒間表示
- シンプルな白背景に"LIFE"ロゴ表示

### 2. 認証システム
- **未認証ユーザー対応**
  - SMS認証（電話番号認証）
  - 検索・閲覧は認証不要
  - アクション（フレンド申請、メッセージ送信）時に認証要求
- **SMS認証機能（詐欺対策）**
  - 電話番号入力
  - 認証コード送信（SMS）
  - 6桁コード確認
  - 本人確認済みバッジ表示
- **プロフィール登録**
  - アイコン設定
  - ニックネーム、年代、性別
  - 都道府県（47都道府県から選択）
  - 自己紹介文
  - よく使う時間帯
  - ギャラリー（6枚まで）

---

## 💬 チャット機能

### 3. チャット一覧画面
- **タブ切り替え**
  - フォロー中のユーザー
  - コミュニティ
- **フレンド検索**
  - 名前、年代でフィルタリング
  - 都道府県でプルダウン検索
  - オンライン/オフライン表示
  - 検索結果からプロフィールモーダル表示
- **プロフィールモーダル**
  - 基本情報表示（年代、地域、活動時間）
  - 本人確認済みバッジ表示
  - 自己紹介（外国人の場合は翻訳ボタン表示）
  - ギャラリー（6枚表示）
  - フレンド申請ボタン（未認証時はSMS認証要求）
  - メッセージ送信ボタン（フレンド後に表示）

### 4. 1対1チャット（ChatDetailScreen）
- メッセージ送信
- タイムスタンプ表示
- **外国人ユーザーとのチャット**
  - メッセージごとに翻訳ボタン表示
  - タップで原文/翻訳切り替え

### 5. コミュニティ検索
- カテゴリー別表示（ゲーム、アニメ、音楽、雑談）
- メンバー数、投稿数表示
- コミュニティ説明

### 6. コミュニティチャット（CommunityDetailScreen）
- グループチャット機能
- メンバー一覧表示
- **外国人メンバーのメッセージ翻訳**
  - メッセージごとに翻訳ボタン
  - 原文/翻訳切り替え

---

## 👤 プロフィール機能

### 7. マイページ
- プロフィール表示
  - アイコン
  - 名前、年代
  - 自己紹介
  - ギャラリー（6枚）
- **メニュー項目**
  - プロフィール編集
  - フレンド申請一覧
  - スタンプ購入
  - 言語設定
  - ログアウト

### 8. プロフィール編集
- 全項目編集可能
- リアルタイム保存

### 9. フレンド申請一覧
- 受信したフレンド申請表示
- 承認/拒否機能（UIのみ）
- 申請者のプロフィール表示

### 10. ユーザープロフィール閲覧
- 他ユーザーのプロフィール表示
- 基本情報（年代、地域、活動時間）
- 本人確認済みバッジ表示
- 自己紹介、ギャラリー
- フレンド状態表示

---

## 🎨 スタンプショップ機能

### 11. スタンプ一覧
- 2列グリッドレイアウト
- スタンプパック情報表示
  - スタンプ名
  - 収録数
  - 必要ポイント
- 12種類のスタンプパック

### 12. スタンプ詳細モーダル
- スタンプパック詳細情報
- スタンププレビュー（最大12個表示）
- ダウンロードボタン
  - 必要ポイント表示
  - ポイント不足時は半透明表示
  - ダウンロード時にポイント消費

### 13. ポイントシステム
- **初期ポイント**: 50pt
- **ポイント表示バー**
  - コインアイコン付き
  - 現在のポイント数表示
  - 「広告を見る」ボタン
- **広告視聴機能**
  - 広告視聴モーダル
  - 3秒間の視聴シミュレーション
  - 視聴完了で10pt獲得
  - 獲得通知表示

## 🌍 国際化（i18n）機能

### 14. 言語設定
- **対応言語**: 日本語、英語
- **言語切り替え**
  - マイページの言語設定メニュー
  - モーダルで言語選択
  - 選択中の言語にチェックマーク表示
- **設定の永続化**
  - localStorageに保存
  - アプリ再起動後も設定維持

### 15. 翻訳機能
- **自動翻訳対象の検出**
  - 外国人ユーザー判定（国フィールド）
  - 日本以外の国のユーザーに翻訳ボタン表示
- **翻訳対象**
  - プロフィールの自己紹介
  - 1対1チャットのメッセージ
  - コミュニティチャットのメッセージ
- **翻訳UI**
  - 翻訳/原文ボタン
  - タップで切り替え
  - 非同期処理でスムーズな表示

### 16. 多言語ユーザーデータ
- **対応言語**: 日本語、英語、中国語、韓国語、フランス語、スペイン語
- **ユーザー数**: 10人（日本5人、外国5人）
- **各ユーザー情報**
  - 名前（現地語）
  - 自己紹介（現地語）
  - 地域（都道府県・市区町村または州・都市）
  - 本人確認ステータス
  - 年代、性別、活動時間

## 🎯 UI/UX機能

### 17. ボトムナビゲーション
- チャット
- マイページ
- 常時表示（各画面下部）

### 18. ヘッダー
- 画面タイトル表示
- 戻るボタン
- 検索ボタン（チャット画面）

### 19. モーダルシステム
- z-indexによる階層管理
  - 通常モーダル: 1000
  - プロフィールモーダル: 1100
  - 認証モーダル: 2000
- 背景タップで閉じる
- スムーズなアニメーション

### 20. ワイヤーフレームデザイン
- シンプルな白黒基調
- 境界線とボックスレイアウト
- アイコンはSVGで統一
- レスポンシブ対応

---

## 📊 データ管理

### 21. モックデータ
- **ユーザーデータ**: 10人
- **コミュニティデータ**: 4件
- **チャットメッセージ**: 日本語版・英語版
- **フレンド申請**: 3件
- **スタンプパック**: 12種類

### 22. 状態管理
- React Hooks（useState）
- グローバル状態なし（propsで伝達）
- localStorageで設定永続化

## 🔧 技術仕様

### 23. フレームワーク・ライブラリ
- **React 18** (UMD builds)
- **Babel Standalone** (JSX変換)
- ブラウザベース（モジュールバンドラー不要）

### 24. 開発環境
- Python HTTP Server（ポート8000）
- ホットリロード非対応
- ブラウザでの直接実行

### 25. ファイル構成
```
src/
├── components/
│   ├── common/          # Icons, Layout
│   ├── screens/         # 各画面コンポーネント
│   └── modals/          # モーダルコンポーネント
├── data/                # mockData.js
├── utils/               # i18n.js
└── styles/              # styles.css
```

## 🎮 使い方

### 起動方法
```bash
cd /Users/takuma/dev/life-mobile
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセス

### 主な操作フロー
1. スプラッシュ画面（2秒）
2. チャット一覧画面が表示
3. 検索でユーザーを探す（国・地域で絞り込み可能）
4. プロフィールを見る（本人確認バッジ確認）
5. フレンド申請（認証が必要）
6. SMS認証で本人確認（詐欺対策）
7. メッセージを送る
8. マイページで設定変更
9. スタンプショップでスタンプ購入
